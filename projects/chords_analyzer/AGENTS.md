Браузерный анализатор аккордов

Требования к коду:

- проект реализуется в директории C:\work\vcs\ailjushkin\projects\chords_analyzer
- проект реализуется в виде приложения на react и удобных sh / bat файлов для запуска проекта для локальной отладки
- проект имеет типовую структуру react - приложения
- код является модульным, логически разделяется по функциям, содержит подробные комментарии и логирование в консоль для легкого дебага
- код пишется так, чтобы можно было менять каждую часть без переписывания всего заново

Требования к проекту:

- лого проекта: guitar.ai, располагается слева сверху в углу страницы
- справа сверху - кнопка загрузить трек
- рядом с кнопкой загрузки трека - список для выбора типа гитары, где по-умолчанию выбрана обычная 6-ти струнная гитара, тип гитары влияет на то, какая табулатура появится в результате анализа аудио-файла
- рядом со списком типов гитар находятся кнопки для транспонирования табулатуры
- ниже посередине область осциллографа, где будет показана осциллограмма проанализированного файла
- ниже осциллограммы кнопки "играть", "стоп"
    - при нажатии на кнопку "играть", она меняется на "пауза", при нажатии на "пауза" - обратно на "играть"
- ниже располагается область распознанных табулатур
- табулатура - это серия карточек с аккордами
- каждый аккорд представляет из себя карточку, в которой описаны гитарные струны и рядом с ними цифры ладов, на которых их нужно зажать, так же там есть такм код, когда именно играется этот аккорд
- при нажатии на аккорд с тайм кодом, индикатор проигрывателя на осциллографе перемещается на указанное в тайм коде место и трек начинает звучать с этого момента
- ниже табулатур находится история проанализированных файлов
- при выборе загруженного файла, во время его анализа, визуализируется workflow анализа этого файла, и показывается, какой сейчас этап

Требования к архитектуре:

- нужно максимально использовать готовые фреймворки и библиотеки из интернета, подходящие под задачу
- задачи анализа аудио файлов пишутся асинхронно, чтобы не подвешивать интерфейс
