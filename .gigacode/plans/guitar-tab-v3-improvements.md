# План доработок GuitarTab.ai v3

## 1. Кнопка очистить историю полностью
- **Текущее состояние**: Кнопка "очистить устаревшие" удаляет только треки с `stale: true`
- **Требование**: Кнопка должна очищать полностью всю базу данных (удалять все треки)
- **Реализация**: Изменить функцию `clearHistory()` чтобы удаляла все треки из IndexedDB

## 2. Белая полоса снизу
- **Проблема**: Снизу экрана отображается белая полоса, фон не сплошной
- **Вероятная причина**: `min-height: 600px` у `.app` или отсутствие `height: 100%` для body/html
- **Решение**: Убрать `min-height` или установить `height: 100vh` для `.app` и `html, body`

## 3. Нажатие на запись истории не загружает в рабочую область
- **Проблема**: Клик по истории не восстанавливает состояние трека (кэш)
- **Реализация**: Добавить обработчик клика на историю, который будет:
  - Получать данные по checksum из IndexedDB
  - Восстанавливать `audioBuffer` из `arrayBuffer`
  - Загружать табулатуры
  - Отображать интерфейс проигрывателя и табулатур

## 4. Осциллограф - более детальное отображение
- **Текущее состояние**: Ширина полосок около `bw * 0.65`
- **Требование**: Сделать пиксели 1-2 пикселя толщиной
- **Решение**: Уменьшить множитель ширины и использовать `stroke` вместо `fillRect` для более точного отображения

## 5. Калибровка тайм-кодов
- **Проблема**: Аккорды не совпадают с моментами звучания в аудио
- **Вероятная причина**: `hopSize = Math.floor(sampleRate * 0.25)` (250мс) слишком большой для точного определения аккордов
- **Решение**: Уменьшить `hopSize` для более частого анализа (например, 50-100мс) или использовать окна с перекрытием
